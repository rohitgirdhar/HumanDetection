Human Detection

This project was done in July-November 2012. I implemented a human detection algorithm, proposed by Zhu, Avidan et.al. in CVPR 2006.

The final report is available here

To run the code

Training (in matlab)

cd HumanDetection/training
det = train_detector(.001,.7,0.9,'test_pos','test_neg')

Now, det is the trained detector

Testing (in matlab)

res = detector('image filename' , det);
DrawRect('image filename', res)

tester functions were used for evaluating the classifier on the standard INRIA pedestrian dataset.
